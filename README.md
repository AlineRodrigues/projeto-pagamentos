# Projeto Pagamentos

Projeto desenvolvido para o Trabalho de Conclusão da Disciplina (Parte 1) da Pós-Graduação em Automação de Testes.

## Tecnologias utilizadas

- Javascript
- Node.js
- Mocha
- Assert

## Funcionalidades

- Realizar pagamentos
- Consultar último pagamento
- Classificar pagamentos por categoria

## Instalação

npm install

## Executar testes

npm test
